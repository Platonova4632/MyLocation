Hello World - Дружелюбное приложение
Имеется строка EditText, где пользователь вводит своё имя, при нажатии кнопки ОК выплывает окно "Close Application..", где Приветствует
нашего пользователя по имени "Hello Username" и есть две кнопки: "Close","Cancel". При нажатии на "Close" - приложение закрывается, "Cancel" - окно убирается. Если пользователь ничего не ввел в строке и нажмёт на кнопку, то вместо окна внизу экрана появится сообщение "Error"
